mer.retimers¶

The retimers module contains classes to convert between different time representations.

Functions

sols_after_mars_year_0(dt)

Compute the number of sols between a datetime and the start of Mars year 0.

sols_between_datetimes(early_dt, later_dt)

Compute the number of sols between two datetimes.

sols_since_datetime(date)

Compute the number of sols between an input datetime and today.
