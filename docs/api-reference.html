API reference¶

The mer package reference.

Earth to Mars¶

Convert Earth datetime objects into Martian times.

datetime_to_fractional_mars_year¶

mer.datetime_to_fractional_mars_year(date)¶

Compute the fractional Mars year of an input datetime.

Parameters

date (datetime.datetime) – Any date.

Raises

TypeError – Raised if date is not a datetime.datetime.

Return type

float

Examples

Convert a datetime to a fractional Mars year.

>>> import datetime, mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0)
>>> mer.datetime_to_fractional_mars_year(date)
35.41260282427384

datetime_to_sol_number¶

mer.datetime_to_sol_number(date)¶

Compute the sol number (day of the Martian year) of an input datetime.

Parameters

date (datetime.datetime) – Any date.

Raises

TypeError – Raised if date is not a datetime.datetime.

Return type

float

Notes

This function begins counting from 0. Beware that some places like LMD use the convention that the new year starts on sol 1.

Examples

Convert a datetime to a sol number of a Mars year.

>>> import datetime, mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0)
>>> mer.datetime_to_sol_number(date)
275.86418326244836

sols_between_datetimes¶

mer.sols_between_datetimes(early_date, later_date)¶

Compute the number of sols between two datetimes.

Parameters

  • early_date (datetime.datetime) – The earlier of the two dates.

  • later_date (datetime.datetime) – The latter of the two dates.

Raises

TypeError – Raised if either early_date or later_date are not a datetime.datetime.

Return type

float

Examples

Compute the number of sols Opportunity was active. I don’t know the hour, minute, or second of the start or end of the mission so I set them to 0.

>>> import datetime, mer
>>> opportunity_start = datetime.datetime(2004, 1, 25, 0, 0, 0)
>>> opportunity_end = datetime.datetime(2018, 6, 10, 0, 0, 0)
>>> mer.sols_between_datetimes(opportunity_start, opportunity_end)
5109.551211085292

Constants¶

A collection of Martian time constants. Note that these can be accessed via the constants module (as shown) or used directly from mer’s namespace (I can’t get the docstrings to render without this; it’s a known Sphinx issue).

mer.constants.date_of_start_of_mars_year_0: datetime.datetime = datetime.datetime(1953, 5, 24, 11, 57, 7, 200011)¶

Time of the start of Mars year 0. This value comes from this article.

mer.constants.martian_sol_length: float = 24.6597¶

Length of a Martian sol [hours]. This value comes from the Mars fact sheet.

mer.constants.seconds_per_sol: float = 88774.92000000001¶

Number of seconds per Martian sol.

mer.constants.sols_per_martian_year: float = 668.5949950729326¶

Number of sols per Martian year.
