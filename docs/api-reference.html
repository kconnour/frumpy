API reference¶

You can find all functions present in the package at this page.

Earth times to Mars times¶

These are a series of functions to convert Earth datetime objects into Martian times.

datetime_to_fractional_mars_year¶

mer.datetime_to_fractional_mars_year(date)¶

Compute the fractional Mars year corresponding to an input datetime.

Parameters

date (datetime.datetime) – Any date.

Raises

TypeError – Raised if date is not a datetime.datetime.

Return type

float

Examples

Convert a datetime into its corresponding fractional Mars year.

>>> import datetime, mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0)
>>> mer.datetime_to_fractional_mars_year(date)
35.41260282427384

datetime_to_sol¶

mer.datetime_to_sol(date)¶

Compute the sol (day of the Martian year) corresponding to an input datetime.

Parameters

date (datetime.datetime) – Any date.

Raises

TypeError – Raised if date is not a datetime.datetime.

Return type

float

Notes

This function begins counting from 0. Beware that some places like LMD use the convention that the new year starts on sol 1.

Examples

Convert a datetime into its corresponding sol.

>>> import datetime, mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0)
>>> mer.datetime_to_sol(date)
275.86418326244836

sols_between_datetimes¶

mer.sols_between_datetimes(early_date, later_date)¶

Compute the number of sols between two datetimes.

Parameters

  • early_date (datetime.datetime) – The earlier of the two dates.

  • later_date (datetime.datetime) – The latter of the two dates.

Raises

TypeError – Raised if either early_date or later_date are not a datetime.datetime.

Return type

float

Examples

Compute the number of sols Opportunity was active. I don’t know the hour, minute, or second of the start or end of the mission so I set them to 0.

>>> import datetime, mer
>>> opportunity_start = datetime.datetime(2004, 1, 25, 0, 0, 0)
>>> opportunity_end = datetime.datetime(2018, 6, 10, 0, 0, 0)
>>> mer.sols_between_datetimes(opportunity_start, opportunity_end)
5109.551211085292

sols_after_mars_year_0¶

mer.sols_after_mars_year_0(date)¶

Compute the number of sols between a datetime and the start of Mars year 0.

Parameters

date (datetime.datetime) – Any date.

Raises

TypeError – Raised if date is not a datetime.datetime.

Return type

float

Examples

Find the number of sols after Mars year 0 that MAVEN arrived at Mars.

>>> import datetime, mer
>>> date = datetime.datetime(2014, 9, 2, 2, 24, 0)
>>> sols = mer.sols_after_mars_year_0(date)
>>> sols
21781.872772174716

Get the Mars year at the arrival date. Note that it’s more efficient to use the built-in function.

>>> sols / mer.sols_per_martian_year
32.57857586833816
>>> mer.datetime_to_fractional_mars_year(date)
32.57857586833816

Mars times to Earth times¶

Convert Martian times to Earth datetimes.

mars_year_to_datetime¶

mer.mars_year_to_datetime(mars_year)¶

Compute the datetime corresponding to an input Mars year.

Parameters

mars_year (float) – Any Mars year.

Raises

  • TypeError – Raised if mars_year is not an int or a float.

  • OverflowError – Raised if mars_year is too far in the past or future. This occurs around -1038 and 4278.

Return type

datetime.datetime

Examples

Convert a Mars year into its corresponding datetime.

>>> import datetime, mer
>>> mars_year = 35.41260282427384
>>> mer.mars_year_to_datetime(mars_year)
datetime.datetime(2020, 1, 1, 0, 0)

Convert sol 300 of Mars year 28 into a datetime.

>>> import datetime, mer
>>> my28_start_date = mer.mars_year_to_datetime(28)
>>> my28_start_date
datetime.datetime(2006, 1, 21, 17, 48, 28, 800011)
>>> seconds_since_year_start = 300 * mer.seconds_per_sol
>>> my28_start_date + datetime.timedelta(seconds=seconds_since_year_start)
datetime.datetime(2006, 11, 25, 23, 43, 4, 800011)

Constants¶

A collection of Martian time constants. Except where noted, these values come directly or indirectly from the Mars fact sheet.

Note that these can be accessed via the constants module (as shown) or used directly from mer’s namespace.

mer.constants.date_of_start_of_mars_year_0: datetime.datetime = datetime.datetime(1953, 5, 24, 11, 57, 7, 200011)¶

Time of the start of Mars year 0. This value comes from this article.

mer.constants.martian_sol_length: float = 24.6597¶

Length of a Martian sol [hours].

mer.constants.seconds_per_sol: float = 88774.92000000001¶

Number of seconds per Martian sol.

mer.constants.sols_per_martian_year: float = 668.5949950729326¶

Number of sols per Martian year.
