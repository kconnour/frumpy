API reference¶

You can find all classes present in the package at this page. In general, all classes have methods to convert to all other time representations.

EarthDatetime¶

class mer.EarthDatetime(dt)¶

Convert Earth datetimes into Martian times.

Parameters

dt (datetime.datetime) – Any datetime.

Raises

TypeError – Raised if dt is not a datetime.datetime.

Notes

dt can be “aware” (have time zone information included with it) or “unaware” (have no associated time zone information). If the input is unaware, it is assumed to be a UTC time and will have that info added to it.

Examples

This object simply accepts datetimes. You can include time zone information as shown below.

>>> import datetime, pytz, mer
>>> dt = datetime.datetime(2020, 1, 1, 0, 0, 0, 0, tzinfo=pytz.UTC)
>>> aware_dt = EarthDatetime(dt)
>>> print(aware_dt)
2020-01-01 00:00:00+00:00

If no time zone information is included, the datetime is assumed to be in UTC.

>>> dt = datetime.datetime(2020, 1, 1, 0, 0, 0, 0)
>>> unaware_dt = EarthDatetime(dt)
>>> print(aware_dt)
2020-01-01 00:00:00+00:00

You can also include non-UTC timezones.

>>> eastern = pytz.timezone('US/Eastern')
>>> dt = datetime.datetime(2020, 1, 1, 0, 0, 0, 0, tzinfo=eastern)
>>> print(EarthDatetime(dt))
2020-01-01 00:00:00-04:56
to_fractional_mars_year()¶

Compute the fractional Mars year corresponding to the input datetime.

Examples

Convert a datetime into its corresponding fractional Mars year.

>>> import datetime, pytz, mer
>>> dt = datetime.datetime(2020, 1, 1, 0, 0, 0, 0, tzinfo=pytz.UTC)
>>> EarthDatetime(dt).to_fractional_mars_year()
35.41260282427384
Return type

float

to_whole_mars_year()¶

Compute the integer Mars year corresponding to the input datetime.

Examples

Convert a datetime into its corresponding “whole” Mars year.

>>> import datetime, mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0)
>>> EarthDatetime(date).to_whole_mars_year()
35
Return type

int

to_sol()¶

Compute the sol (day of the Martian year) corresponding to an input datetime.

Notes

This function begins counting from 0. Beware that some places like LMD use the convention that the new year starts on sol 1.

Examples

Convert a datetime into its corresponding sol.

>>> import datetime, mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0)
>>> EarthDatetime(date).to_sol()
275.86418326244836
Return type

float

sol differences¶

A collection of functions for converting sol differences.

sols_after_mars_year_0¶

mer.sols_after_mars_year_0(dt)¶

Compute the number of sols between a datetime and the start of Mars year 0.

Parameters

dt (datetime.datetime) – Any datetime.

Raises

TypeError – Raised if dt is not a datetime.datetime.

Return type

float

Examples

Find the number of sols after Mars year 0 that MAVEN arrived at Mars.

>>> import datetime, mer
>>> maven_arrival_datetime = datetime.datetime(2014, 9, 2, 2, 24, 0)
>>> mer.sols_after_mars_year_0(maven_arrival_datetime)
21781.872772174716

sols_between_datetimes¶

mer.sols_between_datetimes(early_dt, later_dt)¶

Compute the number of sols between two datetimes.

Parameters

  • early_dt (datetime.datetime) – The earlier of the two datetimes.

  • later_dt (datetime.datetime) – The latter of the two datetimes.

Raises

TypeError – Raised if either early_dt or later_dt are not a datetime.datetime.

Return type

float

Examples

Compute the number of sols Opportunity was active. I don’t know the hour, minute, or second of the start or end of the mission so I set them to 0.

>>> import datetime, mer
>>> opportunity_start = datetime.datetime(2004, 1, 25, 0, 0, 0)
>>> opportunity_end = datetime.datetime(2018, 6, 10, 0, 0, 0)
>>> mer.sols_between_datetimes(opportunity_start, opportunity_end)
5109.551211085292

Constants¶

A collection of Martian time constants. Except where noted, these values come directly or indirectly from the Mars fact sheet.

Note that these can be accessed via the constants module (as shown) or used directly from mer’s namespace.

Time of the start of Mars year 0. This value comes from this article.

mer.constants.martian_sol_length: float = 24.6597¶

Length of a Martian sol [hours].

mer.constants.seconds_per_sol: float = 88774.92000000001¶

Number of seconds per Martian sol.

mer.constants.sols_per_martian_year: float = 668.5949950729326¶
