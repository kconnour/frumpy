convert_to_solar_longitude¶

mer.convert_to_solar_longitude(date)¶

Compute the Martian solar longitude of an input datetime.

Parameters

date (datetime.datetime) – Any date.

Raises

TypeError – Raised if date is not a datetime.datetime.

Return type

float

Examples

Convert a date to solar longitude.

>>> import datetime
>>> import mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0)
>>> mer.convert_to_solar_longitude(date)
128.8354595387973

References

The equation used to convert to Ls can be found in this paper.

convert_to_sol_number¶

mer.convert_to_sol_number(date)¶

Compute the sol number (day of the year) of an input datetime.

Parameters

date (datetime.datetime) – Any date.

Raises

TypeError – Raised if date is not a datetime.datetime.

Return type

float

Notes

This function begins counting from 0. Beware that some places like LMD use the convention that the new year starts on sol 1.

Examples

Convert a date to a sol number from that Mars year.

>>> import datetime
>>> import mer
>>> date = datetime.datetime(2020, 1, 1, 0, 0, 0)
>>> mer.convert_to_sol_number(date)
275.86418551545955

sols_between_dates¶

mer.sols_between_dates(early_date, later_date)¶

Compute the number of sols between two datetimes.

Parameters

  • early_date (datetime.datetime) – The earlier of the two dates.

  • later_date (datetime.datetime) – The latter of the two dates.

Raises

TypeError – Raised if either early_date or later_date are not a datetime.datetime.

Return type

float

Examples

Compute the number of sols Opportunity was active. I don’t know the hour, minute, or second of the start or end of the mission so I set them to 0.

>>> import datetime
>>> import mer
>>> opportunity_start = datetime.datetime(2004, 1, 25, 0, 0, 0)
>>> opportunity_end = datetime.datetime(2018, 6, 10, 0, 0, 0)
>>> mer.sols_between_dates(opportunity_start, opportunity_end)
5109.551211085292
